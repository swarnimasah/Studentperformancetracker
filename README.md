## Project Title
*Student Performance Tracker using Python*

##  Overview of the Project
The Student Performance Tracker is a simple Python-based application designed to help track and evaluate the academic performance of students.  
It allows users to add new students, record marks, calculate averages, and view student performance reports in an organized way.

The system is modular, beginner-friendly, and demonstrates the use of Object-Oriented Programming (OOP) in Python.


##  Features
- Add student details (ID & Name)
- Add multiple marks for each student
- Automatically calculate average marks
- View detailed performance of a selected student
- View all students in the system with marks & averages

## Technologies / Tools Used

*Python 3*

*VS Code* (or any editor)

*Object-Oriented Programming*

*Dictionary-based data storage*

## Steps to Install & Run the Project

### *1. Install Python*

Download Python from:

https://www.python.org/downloads/

### 2. Create the project folder*

Create a folder named: Studentperformancetracker

### 3. Inside the folder, create these files
student.py , storage.py performance.py

### *4. Open the folder in VS Code*
Right-click the folder → Open with VS Code

### *5. Open Terminal*
VS Code → Terminal → New Terminal

### *6. Run the project*
The menu will appear and the program will start running
