## Problem Statement
Educational institutions often need a simple and efficient method to record student marks and calculate performance.  
Manually tracking performance can be time-consuming and error-prone.  
This project provides a lightweight Python-based tool to manage and evaluate student performance quickly.

## Scope of the Project
The project focuses on:

- Maintaining basic student information  
- Adding and storing marks  
- Calculating average performance  
- Displaying individual and group performance  
- A clean, simple user interface for easy use  

The project is intentionally designed to be console-based to keep it simple and educational.

## Target Users
- School teachers  
- Coaching institutes  
- College students  
- Anyone who wants a simple performance tracking program  


## High-Level Features
1. *Add Student*  
   Create a new student record with ID and name.

2. *Record Marks*  
   Add multiple marks for a student.

3. *Calculate Average*  
   Automatically calculates average marks.

4. *View Student Report*  
   Shows all marks and average for a chosen student.

5. *View All Students*  
   Displays every student in the system.

6. *Simple Storage System*  
   Uses a Python dictionary and OOP structure to manage data.
